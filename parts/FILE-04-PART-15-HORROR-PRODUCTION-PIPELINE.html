Part 15 — Horror Production Pipeline (Overview)

यह पूरे स्टूडियो का operating system है — research से monetization तक 17 systems। हर system के 7 हिस्से: Workflow, SOP, Checklist, Tools, Quality Control, Common Mistakes, Optimization Strategy।

01 · Research System

Workflow
  • स्रोत scan: r/nosleep, urban legends, गाँव की अफ़वाहें, बुज़ुर्गों की बातें
  • AI से 10 idea → सबसे डरावना shortlist
  • 100-stories master से dedup check
SOP (Step-by-Step)
  • 1. weekly idea-dump (20 ideas)
  • 2. shortlist 4
  • 3. audience+pillar tag
  • 4. hook test (1 line)
Checklist
  • idea source noted
  • duplicate check done
  • pillar assigned
  • hook drafted
Tools
  • Reddit
  • Google Trends
  • AnswerThePublic
  • Notion/Sheet
Quality Control
  • हर idea का एक-लाइन hook हो
  • pillar balance (urban/rural/folklore/psych)
Common Mistakes
  • सिर्फ़ trending नकल
  • बिना hook के idea
  • एक ही pillar पर ज़्यादा झुकाव
Optimization Strategy
  • evergreen + trend mix
  • best-performing pillar को double करो

02 · Story Development System

Workflow
  • concept → 7-beat outline → full script
  • AI draft + human emotion pass
  • ज़ोर से read-through
SOP (Step-by-Step)
  • 1. outline (गली फॉर्मूला)
  • 2. draft
  • 3. emotion/desi pass
  • 4. hook tighten
  • 5. read aloud
Checklist
  • hook 15s tight
  • 7 beats present
  • unsettling end
  • read-aloud done
Tools
  • Claude/ChatGPT
  • Google Docs
  • Hemingway (clarity)
Quality Control
  • pacing slow-burn
  • fear central, no cheap scares
  • ear-friendly language
Common Mistakes
  • info-dump intro
  • rushed ending
  • किताबी भाषा (वो गली में)
Optimization Strategy
  • हर 40-60s open-loop
  • best scripts के pattern दोहराओ
