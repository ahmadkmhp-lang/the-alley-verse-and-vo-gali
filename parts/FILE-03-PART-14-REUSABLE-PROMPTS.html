Part 14 — Reusable Prompt Library (Overview)

ये 12 templates सभी 100 कहानियों पर काम करते हैं। हर template के 7 हिस्से: Purpose, Inputs, Outputs, Complete Copy-Paste Prompt, Example Usage, Advanced Version, Professional Version। [PLACEHOLDERS] को अपनी कहानी के हिसाब से भरें। Brand tone: Alley Verse = ठंडा/cinematic/English, वो गली = देसी/हिंदी/folklore।

Template 1 — Story Generation Prompt

Purpose

एक premise/idea से पूरा, मौलिक हॉरर स्टोरी-कॉन्सेप्ट बनाना।

Inputs
  • Brand व audience
  • Subgenre
  • एक-लाइन idea/premise
  • Tone
Outputs
  • Title
  • 15-sec Hook
  • Premise
  • Fear mechanism
  • Unsettling ending
Complete Copy-Paste Prompt
Base Prompt
You are a master psychological horror writer for "[BRAND]" (audience: [AUDIENCE]). From this seed: "[IDEA]", generate a Tier-1 horror concept. Subgenre: [SUBGENRE]. Tone: [TONE]. Fear must be psychological, not cheap jump-scares. Deliver: Title, 15-second Hook, full Premise, Fear Mechanism, and an unsettling Ending. Make it culturally authentic and original.
Example Usage

Seed: 'एक लिफ्ट जो रात को एक न-बनी मंज़िल पर रुकती है।' → Title/Hook/Premise/Fear/Ending।

Advanced Version
Advanced (add to base)
...उपरोक्त के साथ: 3 vary किए हुए concepts दो, हर एक का retention-hook अलग; और बताओ कौन-सा thumbnail-friendly है और क्यों।
Professional Version
Professional (add to base)
...Professional: 5 concepts एक table में (Title | Hook | Fear | Why it retains | Thumbnail idea), फिर सबसे viral-योग्य एक को pick करके justify करो (CTR + watch-time logic)।

Template 2 — Script Writing Prompt

Purpose

concept को पूरी narration स्क्रिप्ट (3000+ शब्द) में बदलना।

Inputs
  • Title
  • Hook
  • Premise
  • Fear
  • Ending
  • भाषा/टोन
Outputs
  • Hook (0-15s)
  • 7-beat body
  • narration-ready script
  • scene markers
Complete Copy-Paste Prompt
Base Prompt
Write a complete 3000+ word horror narration script for "[TITLE]" ([BRAND]). Use the 7-beat गली फॉर्मूला: normal life -> hook -> first sign -> rising dread -> reveal -> climax -> unsettling end. Hook: [HOOK]. Premise: [PREMISE]. Fear: [FEAR]. Ending: [ENDING]. Language/tone: [LANG]. Slow burn, use silence and sensory detail, write for the EAR.
Example Usage

Title 'गली नंबर सात' के सभी फ़ील्ड भरकर → पूरी 3000-शब्द हिंदी स्क्रिप्ट।

Advanced Version
Advanced (add to base)
...Advanced: हर 40-60 सेकंड पर एक open-loop/retention-hook जोड़ो; [n] timestamps के साथ scene markers दो; अंत में 3 Shorts-clip suggestions।
Professional Version
Professional (add to base)
...Professional: स्क्रिप्ट + पढ़ने का अनुमानित runtime + WPM pacing notes + जहाँ pause/silence हो वहाँ [PAUSE] markers + voiceover director notes।

Template 3 — Character Creation Prompt

Purpose

कहानी का relatable मुख्य किरदार + entity गढ़ना।

Inputs
  • Title
  • Brand
  • base appearance
  • fear mechanism
Outputs
  • Name/age/occupation
  • flaw
  • emotional arc
  • entity (suggestion only)
Complete Copy-Paste Prompt
Base Prompt
Create the full character profile for the protagonist of "[TITLE]" ([BRAND]). Base look: [LOOK]. Provide name, age, occupation, a relatable trap/flaw, why the audience identifies, and the arc normal->unsettled->afraid->changed. Define the entity rooted in [FEAR] by suggestion only. Realistic, not a stereotype.
Example Usage

'आख़िरी बस अड्डा' के लिए किरदार 'रोहित' का full profile + entity।

Advanced Version
Advanced (add to base)
...Advanced: 2 supporting characters + उनके motives; किरदार-consistency sheet (face/wardrobe/palette) ताकि AI images में identical रहे।
Professional Version
Professional (add to base)
...Professional: किरदार का psychological profile (fear response, decision logic), एक 'AI character reference prompt' (fixed-seed लायक), और continuity rules पूरे episode के लिए।

Template 4 — Scene Breakdown Prompt

Purpose

स्क्रिप्ट को shot-by-shot प्रोडक्शन सूची में तोड़ना।

Inputs
  • script/outline
  • target runtime
Outputs
  • scene list
  • location/time
  • mood
  • key visual
  • sound cue
  • camera move
Complete Copy-Paste Prompt
Base Prompt
Break "[TITLE]" into a scene-by-scene shot list. For each scene: number + duration, location & time, what happens (1-2 lines), mood, key visual, sound design note, camera move. Total ~[RUNTIME]. Mark the HOOK and CLIMAX scenes.
Example Usage

8-15 मिनट की कहानी → 8-12 scenes की shot list।

Advanced Version
Advanced (add to base)
...Advanced: हर scene के लिए AI-image prompt seed + transition type (cut/fade) जोड़ो।
Professional Version
Professional (add to base)
...Professional: पूरी shot list एक table में (Scene | Dur | Location | Action | Mood | Visual | Sound | Camera | Image-prompt) + total runtime calc + asset count।

Template 7 — Animation Prompt Template

Purpose

still images से 2.5D parallax animation बनाना।

Inputs
  • scene layers
  • motion style
Outputs
  • FG/MG/BG/Depth/Fog/Shadow layer specs
  • parallax + zoom instructions
Complete Copy-Paste Prompt
Base Prompt
Animate "[SCENE]" in 2.5D parallax: Foreground [..], Midground [character], Background [..], Depth haze, Fog layer (30-50% opacity, slow drift), Shadow layer (entity, slow approach). Slow zoom 1.0->1.08 over 8-12s, slow push-in, atmospheric motion only. Parallax: FG fast, MG mid, BG slow.
Example Usage

गाँव की गली का दृश्य → 6 layers + parallax।

Advanced Version
Advanced (add to base)
...Advanced: हर layer का displacement map note + fog opacity keyframes + shadow timing।
Professional Version
Professional (add to base)
...Professional: पूरा layer-stack spec (z-depth values) + camera path + render settings + consistency rule (entity हमेशा अधूरा)।

Template 8 — Camera Movement Prompt Template

Purpose

हर beat के लिए dread-बढ़ाने वाला camera move चुनना।

Inputs
  • scene beats
Outputs
  • per-beat camera instruction
Complete Copy-Paste Prompt
Base Prompt
For each beat choose a camera move: Hook=slow push-in on a detail; Rising dread=locked-off static; Reveal=slow tilt/pan to expose; Climax=slight handheld; End=slow pull-back to wide, fade to black. SLOW only — horror lives in patience.
Example Usage

5-beat कहानी → हर beat का camera move।

Advanced Version
Advanced (add to base)
...Advanced: lens (focal length) + DOF + frame composition (rule of thirds / negative space) जोड़ो।
Professional Version
Professional (add to base)
...Professional: पूरी camera sheet (Beat | Move | Lens | DOF | Composition | Duration) + signature brand shot definition।
