Production Pipeline — Overview

यह विभाग पूरे स्टूडियो का operating system है — आइडिया से published वीडियो तक हर चरण के SOP।

Research Workflow

Steps
  • स्रोत: Reddit (r/nosleep), urban legends, गाँव की सच्ची अफ़वाहें, बुज़ुर्गों की कहानियाँ
  • AI से 10 आइडिया → सबसे डरावना चुनें
  • हर आइडिया को 100-stories master से cross-check करें (दोहराव से बचें)
  • टारगेट दर्शक व pillar (urban/rural/folklore/psychological) तय करें

Writing Workflow

Steps
  • Story file से structure लें (गली फॉर्मूला 7 beats)
  • AI draft → खुद emotion + देसी/cinematic touch जोड़ें
  • ज़ोर से पढ़कर जाँचें (narration के लिए लिखा है)
  • Hook 15s tight; अंत बेचैन छोड़ने वाला

Image Workflow

Steps
  • Character reference image (fixed seed) पहले बनाएँ
  • AI-IMAGE-PROMPTS file से 7 prompts चलाएँ
  • Consistency check: चेहरा/पहनावा/palette हर image में एक जैसा
  • Entity हमेशा आधा छाया में

Video Workflow

Steps
  • AI-VIDEO-PROMPTS file के 5 beats generate करें
  • कम motion, slow push-in; तेज़ pan नहीं
  • 2.5D animation file से parallax/zoom लगाएँ
  • Lighting व camera direction फॉलो करें

Editing Workflow

Steps
  • Voiceover + visuals sync
  • Slow zoom/parallax + fog overlay + color grade (palette)
  • Sound mix: VO साफ़, music low; scare से पहले silence
  • 3 Shorts काटें (re-purpose)

Publishing Workflow

Steps
  • Title + thumbnail (A/B) — SEO file से
  • Description + timestamps + hashtags + tags + सही playlist
  • End screen + cards + captions
  • सही समय पर schedule; पहले 1 घंटे comments का जवाब + share

Analytics Workflow

Steps
  • CTR (टारगेट 5%+) — कम हो तो thumbnail/title बदलें
  • Avg View Duration (50%+) + retention graph — drop वाली जगह सुधारें
  • पहले 30s retention (70%+) — hook tighten
  • साप्ताहिक: best वीडियो जैसा और; मासिक: best pillar पर फोकस
